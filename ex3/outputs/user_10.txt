## Regularization בלמידת מכונה - סקירה מקיפה עם דוגמאות קוד

Regularization היא טכניקה חשובה בלמידת מכונה שנועדה למנוע overfitting - מצב שבו מודל לומד את הנתונים בצורה מושלמת מדי, כך שהוא לא מצליח להכליל (generalize) לדאטה חדשה. Regularization מוסיפה קנס למודל על מורכבותו, ובכך מעודדת אותו ללמוד מודל פשוט יותר שמסוגל להכליל טוב יותר. הנה פירוט של הסוגים השונים של Regularization:

**I. L1 Regularization (Lasso):**

*   **עיקרון:** מוסיף קנס פרופורציונלי לסכום הערכים המוחלטים של המשקלים במודל.
*   **נוסחה:** Loss = Loss(data) + λ * Σ |wᵢ| (כאשר wᵢ הם המשקלים ו-λ הוא פרמטר ה-Regularization)
*   **יתרונות:**
    *   מבצע בחירת תכונות (Feature Selection): מכריח משקלים של תכונות לא חשובות להיות אפס, ובכך מפחית את מספר התכונות במודל.
    *   מייצר מודל פשוט יותר וקל יותר לפירוש.
*   **חסרונות:**
    *   לא מתאים תמיד לכל סוגי הנתונים.
    *   יכול להיות קשה לבחור את הערך האופטימלי של λ.
*   **דוגמת קוד (Python - scikit-learn):**

    ```python
    from sklearn.linear_model import Lasso
    from sklearn.datasets import make_classification

    # יצירת נתונים סינתטיים
    X, y = make_classification(n_samples=100, n_features=10, random_state=42)

    # הגדרת מודל Lasso עם פרמטר regularization של 0.1
    lasso = Lasso(alpha=0.1)

    # אימון המודל
    lasso.fit(X, y)

    # הצגת המשקלים של המודל
    print(lasso.coef_)
    ```

**II. L2 Regularization (Ridge):**

*   **עיקרון:** מוסיף קנס פרופורציונלי לסכום ריבועי המשקלים במודל.
*   **נוסחה:** Loss = Loss(data) + λ * Σ wᵢ²
*   **יתרונות:**
    *   מפחית את השפעת תכונות בעלות שונות גבוהה.
    *   מייצר מודל יציב יותר.
*   **חסרונות:**
    *   לא מבצע בחירת תכונות. כל התכונות מקבלות משקל (אפילו אם קרוב לאפס).
    *   יכול להיות קשה לבחור את הערך האופטימלי של λ.
*   **דוגמת קוד (Python - scikit-learn):**

    ```python
    from sklearn.linear_model import Ridge
    from sklearn.datasets import make_classification

    # יצירת נתונים סינתטיים
    X, y = make_classification(n_samples=100, n_features=10, random_state=42)

    # הגדרת מודל Ridge עם פרמטר regularization של 0.1
    ridge = Ridge(alpha=0.1)

    # אימון המודל
    ridge.fit(X, y)

    # הצגת המשקלים של המודל
    print(ridge.coef_)
    ```

**III. Elastic Net Regularization:**

*   **עיקרון:** שילוב של L1 ו-L2 Regularization. מוסיף קנס שמורכב מסכום של הערכים המוחלטים של המשקלים וסכום ריבועי המשקלים.
*   **נוסחה:** Loss = Loss(data) + λ₁ * Σ |wᵢ| + λ₂ * Σ wᵢ²
*   **יתרונות:**
    *   משלב את היתרונות של L1 ו-L2 Regularization.
    *   מבצע בחירת תכונות ומייצר מודל יציב יותר.
*   **חסרונות:**
    *   יכול להיות קשה לבחור את הערכים האופטימליים של λ₁ ו-λ₂.
*   **דוגמת קוד (Python - scikit-learn):**

    ```python
    from sklearn.linear_model import ElasticNet
    from sklearn.datasets import make_classification

    # יצירת נתונים סינתטיים
    X, y = make_classification(n_samples=100, n_features=10, random_state=42)

    # הגדרת מודל ElasticNet עם פרמטר regularization של 0.1 ו-l1_ratio של 0.5
    elastic_net = ElasticNet(alpha=0.1, l1_ratio=0.5)

    # אימון המודל
    elastic_net.fit(X, y)

    # הצגת המשקלים של המודל
    print(elastic_net.coef_)
    ```

    *   מפחית את ה overfitting.

    *   דורש התאמת פרמטר ה-Dropout.
    *   יכול להאריך את זמן האימון.

**V. Early Stopping:**

*   **עיקרון:** טכניקה של Regularization שמשמשת ברשתות נוירונים ובמכונות וקטור תומכות (SVM). במהלך האימון, עוצרים את האימון כאשר הביצועים של המודל על סט הvalidation מתחילים להשתפל. זה מונע מהמודל ללמוד בצורה מושלמת מדי את הנתונים ומייצר מודל שמסוגל להכליל טוב יותר לדאטה חדשה.

**בחירת פרמטר ה-Regularization (λ):**
