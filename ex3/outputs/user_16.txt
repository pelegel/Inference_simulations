## Regularization בלמידת מכונה: מדריך מקיף

Regularization היא טכניקה חשובה בלמידת מכונה שמטרתה למנוע overfitting - מצב שבו המודל לומד את הנתונים בצורה מושלמת מדי, עד כדי שהוא מאבד את היכולת שלו להכליל לנתונים חדשים. Regularization מוסיפה עונש למודל על מורכבותו, ובכך מעודדת אותו ללמוד מודלים פשוטים יותר שיכולים להכליל טוב יותר. הנה סקירה מקיפה של סוגי ה-regularization השונים:

**1. L1 Regularization (Lasso)**

*   **העיקרון:** מוסיפה לloss function סכום של הערכים המוחלטים של המשקלים של המודל.
*   **נוסחה:**
    `Loss = Original Loss + λ * Σ |wᵢ|`
    כאשר:
    *   `Loss` הוא הloss function המקורי.
    *   `λ` (למבדה) הוא פרמטר הregularization שמגדיר את עוצמת העונש.
    *   `wᵢ` הוא המשקל ה-i של המודל.
*   **יתרונות:**
    *   **Feature Selection:** נוטה לדחוף משקלים של פיצ'רים לא רלוונטיים לאפס, ובכך מבצעת בחירה של פיצ'רים באופן אוטומטי.
    *   פשוטה יחסית ליישום.
*   **חסרונות:**
    *   לא תמיד יעילה כמו L2 regularization.
    *   ייתכן שפיתון של פיצ'רים לא רלוונטיים לא תיתן שיפור משמעותי בדיוק המודל.
*   **דוגמת קוד (Python):**

    ```python
    import numpy as np
    from sklearn.linear_model import Lasso
    from sklearn.datasets import make_regression

    # Create synthetic data
    X, y = make_regression(n_samples=100, n_features=10, noise=0.1)

    # Initialize Lasso model
    lasso = Lasso(alpha=0.1)  # α is lambda

    # Train the model
    lasso.fit(X, y)

    # Print coefficients
    print(lasso.coef_)
    ```

**2. L2 Regularization (Ridge)**

*   **העיקרון:** מוסיפה לloss function סכום של ריבועי המשקלים של המודל.
*   **נוסחה:**
    `Loss = Original Loss + λ * Σ wᵢ²`
    כאשר:
    *   `Loss` הוא הloss function המקורי.
    *   `λ` (למבדה) הוא פרמטר הregularization שמגדיר את עוצמת העונש.
    *   `wᵢ` הוא המשקל ה-i של המודל.
*   **יתרונות:**
    *   פשוטה ליישום.
    *   יעילה במניעת overfitting.
    *   מפחיתה את הרגישות של המודל לשינויים קטנים בנתונים.
*   **חסרונות:**
    *   לא מבצעת בחירה של פיצ'רים. כל הפיצ'רים מקבלים משקל (אפילו אם הוא קטן מאוד).
*   **דוגמת קוד (Python):**

    ```python
    import numpy as np
    from sklearn.linear_model import Ridge
    from sklearn.datasets import make_regression

    # Create synthetic data
    X, y = make_regression(n_samples=100, n_features=10, noise=0.1)

    # Initialize Ridge model
    ridge = Ridge(alpha=0.1) # α is lambda

    # Train the model
    ridge.fit(X, y)

    # Print coefficients
    print(ridge.coef_)
    ```

**3. Elastic Net Regularization**

*   **העיקרון:** שילוב של L1 ו-L2 regularization.
*   **נוסחה:**
    `Loss = Original Loss + λ₁ * Σ |wᵢ| + λ₂ * Σ wᵢ²`
    כאשר:
    *   `Loss` הוא הloss function המקורי.
    *   `λ₁` ו-`λ₂` הם פרמטרי regularization שמגדירים את עוצמת העונש של L1 ו-L2, בהתאמה.
*   **יתרונות:**
    *   משלבת את היתרונות של L1 ו-L2 regularization.
    *   יכולה להיות יעילה במיוחד במקרים שבהם יש קורלציה גבוהה בין הפיצ'רים.
*   **חסרונות:**
    *   מורכבת יותר ליישום.
    *   צריך לבחור שני פרמטרים של regularization (λ₁ ו-λ₂).
*   **דוגמת קוד (Python):**

    ```python
    import numpy as np
    from sklearn.linear_model import ElasticNet
    from sklearn.datasets import make_regression

    # Initialize ElasticNet model
    elastic_net = ElasticNet(alpha=0.1, l1_ratio=0.5) # α is lambda, l1_ratio determines L1/L2 mix

    # Train the model
    elastic_net.fit(X, y)

    # Print coefficients
    print(elastic_net.coef_)
    ```
