## תהליך ייצור בירה: מהשדה לכוס - פירוט מקיף

ייצור בירה הוא תהליך עתיק יומין המשלב בין מדע לאומנות. למרות שקיימות וריאציות רבות בין מבשלות שונות, התהליך הבסיסי נותר דומה. הנה פירוט מקיף של שלבי ייצור הבירה:

**1. הכנת המלט (Malting):**

*   **בחירת הדגן:** לרוב משתמשים בחיטה או בשעורה, אך ניתן להשתמש גם בדגנים אחרים כגון תירס ואורז. בחירת הדגן משפיעה על הטעם והאופי של הבירה.
*   **הרטבה:** השעורה מושרית במים למשך מספר ימים כדי לעורר את תהליך הנביטה.
*   **נביטה:** השעורה מונחת על רצפה ומונבת בתנאים מבוקרים של טמפרטורה ולחות. במהלך הנביטה מופרשות אנזימים הממפרקים את עמילן הדגן לסוכרים פשוטים.
*   **ייבוש:** השעורה המנובטת מיובשת בחום גבוה כדי לעצור את תהליך הנביטה ולשמר את האנזימים. הטרנספורמציה הזו הופכת את השעורה ל"מלט". צבע המלט משפיע על צבע הבירה - מלט כהה תורם לבירה כהה יותר.

**2. הלעטה (Mashing):**

*   **טחינה:** המלט נטחן לשברים קטנים כדי להגדיל את שטח הפנים שלו ולהקל על הפיכת העמילן לסוכרים.
*   **הלעטה:** המלט הטחון מערבבים עם מים חמים בטמפרטורה מבוקרת במשך שעות. הטמפרטורה משפיעה על הפעילות של האנזימים ועל סוג הסוכרים שנוצרים. תהליך זה הופך את העמילן במלט לסוכרים מסיסים כמו מלטוז, שמשמשים כמרכיב המזון עבור השמרים.
*   **השפית (Lautering):** המלט המרוסק והמים החמים מועברים דרך מסננת כדי להפריד בין התירוש (wort) – הנוזל העשיר בסוכרים - לבין שאריות המלט הלא מסיסות (grain husks). המים שוטפים את שאריות המלט כדי למצות את כל הסוכרים הניתנים להמסה.

**3. הרתחה (Boiling):**

*   **הרתחה:** התירוש מרתיחים במשך שעה עד שעה וחצי. ההרתחה מטרתה לעקר את התירוש ולהרסן חלבונים לא רצויים. בנוסף, במהלך ההרתחה מוסיפים כשות (hops) - צמח מר שמוסיף לבירה טעם, ארומה ומשמש כחומר משמר טבעי. כמות הכשות וזמני הוספתה משפיעים על האופי של הבירה.
*   **צינון:** לאחר ההרתחה התירוש מצונן במהירות כדי למנוע התפתחות של חיידקים לא רצויים.

**4. תסיסה (Fermentation):**

*   **הוספת שמרים:** לאחר הצינון מוסיפים לתירוש שמרים. השמרים ניזונים מהסוכרים המסיסים ומוציאים אלכוהול ופחמן דו חמצני כתופעת לוואי.
*   **סוגי שמרים:** קיימים שני סוגי שמרים עיקריים המשמשים לייצור בירה:
    *   **שמרים עיליים (ale yeast):** מתסיסים בטמפרטורה גבוהה יחסית (18-25 מעלות צלזיוס) ויוצרים טעמים פירותיים ומורכבים. הבירות המיוצרות באמצעות שמרים עיליים נקראות "אל".

*   **הקרמה (Carbonation):** לבירה מוסיפים פחמן דו חמצני כדי ליצור את הגיזוז. ניתן להוסיף פחמן דו חמצני באופן מלאכותי או לאפשר לו להיווצר באופן טבעי במהלך תהליך התסיסה הסופית בבקבוק או בחבית.

*   **הבקבוק/חבית:** הבירה מבוקבקת או מוכנסת לחביות. תהליך הקיבוע חייב להיות סטרילי כדי למנוע זיהומים.

import React, { useState, useEffect } from 'react';
import firebaseConfig from './firebaseConfig';
import { initializeApp } from "firebase/app";
import { getFirestore, collection, query, orderBy, onSnapshot } from "firebase/firestore";
import ChatForm from './components/ChatForm';
import ChatMessages from './components/ChatMessages';

const app = initializeApp(firebaseConfig);
const db = getFirestore(app);

function App() {
  const [messages, setMessages] = useState([]);

  useEffect(() => {
    const collectionRef = collection(db, 'messages');
    const q = query(collectionRef, orderBy('timestamp'));

    const unsubscribe = onSnapshot(q, (snapshot) => {
      const newMessageArray = snapshot.docs.map((doc) => ({ ...doc.data(), id: doc.id }));
      setMessages(newMessageArray);
    });

    return () => unsubscribe(); // Clean up listener
  }, []);

  return (
    <div className="App">
      <h1>Chat App</h1>
      <ChatMessages messages={messages} />
      <ChatForm messages={messages} setMessages={setMessages} />
