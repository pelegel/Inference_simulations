    *   **השרייה:** טבילת הגרעינים במים למשך מספר שעות כדי להתחיל הנביטה.
    *   **נביטה (Germination):** הנחת הגרעינים בתנאים מבוקרים של טמפרטורה ולחות כדי להפעיל את האנזימים הפנימיים שלהם. האנזימים האלה יהיו אחראים להמרה של עמילנים לסוכרים.

*   **מים:** המים מהווים כ-90% מנפח הבירה. איכות המים חשובה מאוד לטעם הסופי של הבירה.
*   **שמרים (Yeast):** השמרים הם מיקרואורגניזמים שאוכלים את הסוכרים ויוצרים אלכוהול ופחמן דו חמצני. ישנם שני סוגים עיקריים של שמרים:

*   **קירור (Cooling):** קירור המשמש לטמפרטורה המתאימה להוספת השמרים.

```javascript
import { getAuth, onAuthStateChanged } from "firebase/auth";
import { getFirestore, collection, addDoc, getDocs, orderBy, query } from "firebase/firestore";

import styled from 'styled-components';

const ChatContainer = styled.div`

const MessagesContainer = styled.div`
  flex: 1;
  overflow-y: auto;

const MessageContainer = styled.div`
  padding: 8px 12px;
  border-radius: 8px;
  margin-bottom: 8px;
`;

const MessageText = styled.span`
  font-size: 16px;
`;

const InputContainer = styled.div`
  padding: 10px;
  border-top: 1px solid #ccc;
`;

const InputField = styled.input`
  padding: 8px;
  border: none;
  border-radius: 4px;
`;

const SendButton = styled.button`
  padding: 8px 12px;
  border: none;
  background-color: #007bff;
  color: white;
  border-radius: 4px;
  cursor: pointer;
`;

const ChatPage = () => {
  const [user] = useAuthState(getAuth());
  const [messages, setMessages] = useState([]);
  const [newMessage, setNewMessage] = useState('');
  const messagesRef = useRef(null);

  const db = getFirestore();
  const messagesCollection = collection(db, "chats");

  useEffect(() => {
    if (user) {
      const q = query(messagesCollection, orderBy("createdAt", "asc"));
      getDocs(q).then((querySnapshot) => {
        const newMessages = querySnapshot.docs.map(doc => ({
          id: doc.id,
          ...doc.data(),
        }));
        setMessages(newMessages);
      });
    }
  }, [user]);

  const sendMessage = async () => {
    if (newMessage.trim() !== '') {
      const newMessageData = {
        sender: user.email,
        text: newMessage,
        createdAt: new Date(),
      };
      await addDoc(messagesCollection, newMessageData);
      setNewMessage('');
      messagesRef.current.scrollTo({ behavior: 'smooth', block: 'end' });
    }
  };

  return (
    <ChatContainer>
      <MessagesContainer ref={messagesRef}>
        {messages.map((message) => (
          <MessageContainer key={message.id}>
            <MessageText>{message.text} - {message.sender}</MessageText>
          </MessageContainer>
        ))}
      </MessagesContainer>
      <InputContainer>
        <InputField
          type="text"
          value={newMessage}
          onChange={(e) => setNewMessage(e.target.value)}
        />
        <SendButton onClick={sendMessage}>Send</SendButton>
      </InputContainer>
    </ChatContainer>
  );
};

export default ChatPage;
```

## Regularization בלמידת מכונה: מדריך מקיף

Regularization היא טכניקה חשובה בלמידת מכונה שמטרתה למנוע "התאמת יתר" (Overfitting) - מצב בו המודל לומד את נתוני האימון בצורה מושלמת מדי, אך מתקשה להכליל לנתונים חדשים. טכניקות regularization מוסיפות "עונש" (penalty) למודל על מורכבות יתר, ובכך מעודדות אותו ללמוד דפוסים כלליים יותר.

**1. סוגי Regularization:**

    *   **נוסחה:**  `Cost Function = Loss Function + λ * Σ |βi|`
        *   `λ` (Lambda): היפר-פרמטר שקובע את עוצמת ה-regularization.
        *   `βi`: מקדמי המודל.
    *   **יתרונות:**
        *   **Feature Selection:** יכולה להוביל להסרת מקדמים ששווים לאפס, ובכך לבחור תכונות רלוונטיות בלבד.
        *   **מודל פשוט יותר:**  מוביל למודל עם פחות מקדמים, ולכן קל יותר להבין ולפרש.
    *   **חסרונות:**

    *   **דוגמת קוד (Python עם Scikit-Learn):**

        ```python
        from sklearn.linear_model import Lasso
        from sklearn.model_selection import train_test_split
        import numpy as np

        # נתונים לדוגמה
        X = np.random.rand(100, 5)
        y = np.random.rand(100)

        # חלוקה לנתוני אימון ובדיקה
        X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

        # יצירת מודל Lasso
        lasso = Lasso(alpha=0.1)  # α הוא הלמבדה

        # אימון המודל

        # חיזוי על נתוני הבדיקה
        y_pred = lasso.predict(X_test)
        ```

    *   **נוסחה:**  `Cost Function = Loss Function + λ * Σ βi²`

        *   `βi`: מקדמי המודל.

        ```python
        X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
        ```

*   **משפט מנהלי:** כללים המסדירים את פעולות הרשות המבצעת (הממשלה).
*   **הסכמים בינלאומיים:** הסכמים שנחתמו עם מדינות אחרות.

*   **בתי משפט השלום:** עוסקים בתביעות אזרחיות ופליליות קטנות, כמו תביעות כספיות בסדר גודל נמוך, תביעות נזיקין קלות, עבירות תעבורה ועוד.

*   **בית הדין לערעורים פליליים:** ערעורים על פסקי דין של בתי המשפט המחוזיים בעבירות פליליות.

*   **פסק דין:** בית המשפט מכריע בסכסוך ומוציא פסק דין.
*   **ערעור:** צד אחד יכול לערער על פסק הדין לבית משפט גבוה יותר.

**5. עקרונות יסוד:**
